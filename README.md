## Excercice réalisé avec des étudiants B3 Développeur Web J1/J2

### Suite à une première journée de découverte de NodeJS Vanilla nous avons découvert ExpressJS.

 - Mise en place d'un serveur basé sur ExpressJS
 - Création de routes pour une API sur une structure permettant une approche modulaire et un routage vers un moteur de rendu
 - Intégration de BCrypt pour le hachage de mot de passe
 - Intégration de JWT pour l'authentification
 - Intégration de Mongoose pour communiquer avec une base de donnée MongoDB
 - Intégration de Swagger pour la documentation
 - Création de routes pour l'authentification
 - Création de route CRUD pour un système de post
 - Création d'un gestionnaire d'erreur
 - Mise en place d'un plan de test
 - Installation d'outils d'amélioration du code
 - Configuration multi environnement
